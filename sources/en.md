# EN source

Первый планируемый источник: `ESDB/SCOWL`.

Source URL:

```text
https://github.com/en-wl/wordlist
```

Для первой версии приложения используется только отфильтрованный список lemma,
подходящих для игровой проверки валидности:

- lowercase `a-z`
- длина от 3 символов
- без дефисов
- без апострофов
- без точек
- без цифр
- без спецсимволов

Этот репозиторий не хранит upstream checkout. Локальный checkout источника
нужен только на машине, где запускается generator.
